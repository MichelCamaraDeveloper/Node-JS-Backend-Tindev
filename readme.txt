Olá!

Este projeto Backend foi desenvolvido em Node JS.
Fruto de um aprendizado realizado na semana Omnistack 8.0 da Rocketseat.

Aqui está desenvolvido a logica do sistema Tindev, um "Tinder" para desenvolvedores.
- Possue os módulos Model e Controller de um MVC.
- Definição de rotas de acesso pelos paramentos HTTP.
Este sistema cadastra um desenvolvedor caso ele ainda não esteja contido na base de dados MongoDB.
O sistema pega o nome, informado pelo usuário, do Github, trás os dados dessa pessoa atraves da API free do Github
e salva na base de dados caso ainda não esteja contigo.
Também é possivel fazer o like e dislike entre desenvolvedores, assim como o match entre desenvolvedores.

Para essa parte do projeto eu usei as seguintes ferramentas:
- Node JS;
- Yarn (
    - express;
    - nodemon;
    - mongoose;
    - axios;
    - cors;
);
- Visual Code;
- MongoDB Atlas;
- MongoDB Compass;
- Insomnia;
- API Github;
